How to install

One signature, every app. The same signature works in Apple Mail, Gmail, Outlook and everything else. How you install it depends on the app: Mac Apple Mail uses the .mailsignature file, everywhere else you copy and paste. Pick your app below.

macOS Mail strips HTML pasted into its signature editor, so the reliable method is to create a placeholder signature, then replace the file on disk.

  1. In Mail, open Settings → Signatures. Pick your account, click +, type any placeholder text (e.g. "temp"), and give it a name. This creates the file you will edit.
  2. Quit Mail completely with Cmd+Q — not just closing the window.
  3. In Finder press Cmd+Shift+G and paste ~/Library/Mail/V10/MailData/Signatures/. The V10 may differ — open the highest-numbered V folder you have.
  4. Sort by Date Modified. The newest .mailsignature file (its name is a long random ID) is the placeholder you just made.
  5. From your card, download the .mailsignature file. Open it and the placeholder file in a plain-text editor (TextEdit set to plain text, or VS Code).
  6. Copy everything from your downloaded file and paste it over the entire contents of the placeholder file. Save, keeping the .mailsignature extension and the original ID filename.
  7. Select the file in Finder, press Cmd+I, and tick Locked.
  8. Reopen Mail and select your signature for the account.
The Locked step is not optional. Without it, Mail or iCloud sync rewrites the file on next launch and your signature reverts to the placeholder. If that happens, redo steps 5–7 and confirm Locked is checked.

Know this up front: when you paste, the signature may come in looking wrong. That is expected, and one undo step fixes it. Do not try to fix the layout by hand.

  1. Open the Open link on your card in Safari. Press and hold on the signature, tap Select All, then Copy. Use Select All rather than dragging the blue handles — a partial selection is the usual cause of a bad paste.
  2. Go to Settings → Mail → Signature and delete anything already in the box.
  3. Tap in the box and Paste. If it comes in looking right, logo and all, you are done — tap the back arrow to save. If it looks wrong, plain text or a broken layout, leave it as is and go to the next step.
  4. Shake the phone. A prompt reading Undo Change Attributes appears — tap Undo. That strips the formatting iOS added on paste and restores the original. Tap back to save.
The shake in step 4 needs Shake to Undo switched on, which it usually is by default. If shaking does nothing, turn it on here and try again.
Using the Gmail app on your phone? Follow the Gmail tab instead — same signature, just pasted into Gmail's settings.

Copy your signature from your card, then add it to Gmail.

  1. On your card, click Copy (or open the Open link, then Select All with Cmd/Ctrl+A and Copy).
  2. In Gmail, go to Settings (gear) → See all settings → General → Signature.
  3. Click Create new, name it, and paste into the editor box.
  4. Under Signature defaults, set it for new emails and for replies/forwards.
  5. Scroll to the bottom and click Save Changes — easy to miss, and nothing saves without it.
Gmail renders the embedded logo fine. Just remember to click Save Changes at the bottom — nothing saves without it.

Outlook on the web is the most reliable; the desktop apps can be inconsistent with HTML signatures.

  1. On your card, click Copy (or open the Open link, Select All and Copy).
  2. Outlook on the web: Settings (gear) → Mail → Compose and reply → Email signature. Paste, name it, set it as default for new messages and replies, and Save.
  3. Outlook desktop: Options → Mail → Signatures (Windows) or Preferences → Signatures (Mac) → New, then paste.
If the desktop app drops the logo or mangles the layout, use Outlook on the web instead — it handles the embedded image reliably.
