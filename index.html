How to install

  1. In Mail, go to Settings → Signatures. Create a new signature (pick any account), type a placeholder word, and name it. This creates the file you will edit.
  2. Quit Mail completely (Cmd+Q).
  3. In Finder press Cmd+Shift+G and go to ~/Library/Mail/V10/MailData/Signatures/ (the V number may differ — choose the highest).
  4. Sort by Date Modified. The newest .mailsignature file is the placeholder you just made.
  5. Download your .mailsignature file from your card above. Open both it and the placeholder file in a plain-text editor.
  6. Replace the entire contents of the placeholder file with the contents of your downloaded file, and save (keep the .mailsignature extension and the original filename).
  7. Select the file in Finder, press Cmd+I, and check Locked. This stops Mail from overwriting it.
  8. Reopen Mail and select your signature for the account.
The Locked step is essential. Without it, Mail (or iCloud sync) can rewrite the file on relaunch and your edit reverts.
